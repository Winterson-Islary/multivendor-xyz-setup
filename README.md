# Multivendor System
Trying to build a simple multivendor store for college where departments and college authorities will be able to list unused or old products or anything for that matter, for sale.

## Goals
	1. Learn about multivendor systems.
	2. Learn about Reservation or Locking systems to maintain inventory consistency.
	3. Build a proper asynchronous system. (sub-tasks to be constructed)
	4. Learn more about Idempotency keys (for safe retries, and prevention of double spends).
